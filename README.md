## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ashraf56/task-management-system.git
   ```

2. Navigate to the project directory:

   ```bash
   cd your-repo
   ```

3. Install dependencies using npm:

   ```bash
   npm install
   ```

   Or using yarn:

   ```bash
   yarn
   ```

4. Usage:

 Run the Vite development server:


 ```bash
   npm run dev
   ```
Or with yarn:

 ```bash
  yarn dev
   ```
